# EMA Crossover Strategy

This is a basic trend-following trading strategy using:

- 9 EMA
- 21 EMA

## Logic
- Buy when EMA 9 crosses above EMA 21
- Sell when EMA 9 crosses below EMA 21

## Notes
- Not optimized
- Educational purpose only
- Realistic results (~50–60% win rate)

## Tech
- Python
- Pandas
